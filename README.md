# 🌍 Ceylon Wonders - Sri Lanka Travel Platform

**Ceylon Wonders** is a full-stack web application designed to showcase the beauty of Sri Lanka. It provides a platform for travelers (both locals and foreigners) to discover hidden gems, plan their itineraries, and manage bookings across different provinces of the island.

## 🚀 Features

* **User Authentication:** Secure Sign-up and Login using JWT.
* **Booking System:** Seamlessly book tours and accommodation.
* **Province-based Search:** Explore Sri Lanka through an interactive province-wise categorization.
* **Dynamic Dashboard:** Manage your profile and upcoming trips.
* **Multi-Currency Support:** View prices in LKR and USD.

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Mongoose ODM)
* **Security:** Bcryptjs for password hashing & JWT for session management.
* **Frontend:** React/HTML/CSS (Tailwind CSS)
